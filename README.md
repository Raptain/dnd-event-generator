Веб-приложение для мастеров Dungeons & Dragons. Генерирует случайные события для пустынной локации из базы данных 100 уникальных встреч.
 В дальнейшем планируется больше
[Открыть генератор](https://dnd-event-generator.onrender.com)

Возможности

 100 уникальных событий
 Рейтинг опасности от 1 до 10
 Наградыпрописаны награды и дальнейшее развитие сюжета как подсказка для мастера
 Адаптивный дизайн — работает на телефонах и планшетах
 Мгновенная генерация по нажатию кнопки
 
 Технологии
Backend: Python 3.11 + Flask 3.0.3
Frontend: HTML5, JavaScript
База данных: Excel (.xlsx) + openpyxl
Деплой: Render.com
Версионный контроль: Git + GitHub
